# iAccelerator Phase 7 Rwanda - Development Plan

## Overview
A comprehensive digital platform for the iAccelerator Programme Phase 7, focusing on Adolescent Sexual and Reproductive Health (ASRH), HIV prevention, menstrual health, and mental health in Rwanda.

## 1. Project Infrastructure
- **Framework**: React 18 + Vite + Tailwind CSS.
- **Backend/DB**: Firebase (Firestore) for application submissions and community feedback.
- **AI Integration**: Gemini API for an anonymous, safe-space Health Assistant.
- **Styling**: Modern, youth-centric design using the "Youthful & Vibrant" recipe from our design guidelines.

## 2. Core Features
- **Landing Page**: Interactive statistics and program overview (based on RDHS 2025 and EICV7 2024 data).
- **Resource Center**: Digital library covering ASRH, Menstrual Hygiene, and Mental Health.
- **AI Health Companion**: Anonymous Q&A powered by Gemini to break barriers and taboos.
- **Entrepreneurship Portal**: Secure submission form for phase 7 applicants.
- **Community Forum**: Safe discussion space for peer support.

## 3. Database Schema (Firestore)
- `applications`: Stores innovative solutions from social entrepreneurs.
- `resources`: Metadata for educational content.
- `forum_posts`: Community discussion threads.

## 4. Implementation Steps
1. **Foundation**: Setup Firebase and basic routing.
2. **UI/UX Core**: Create the vibrant landing page and resource sections.
3. **AI Integration**: Implement the Gemini Health Assistant.
4. **Functionality**: Build the application form with real-time validation.
5. **Testing**: Integrated unit and functional tests for the portal.

## 5. Timeline (Today)
- **Morning**: Infrastructure and UI/UX skeleton.
- **Afternoon**: AI Assistant and Database logic.
- **Evening**: Testing, verification, and final submission.
